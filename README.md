# Cosa Nostra Sandwichería

Sitio web moderno y profesional para Cosa Nostra Sandwichería, ubicada en Talca, Chile.

## 🚀 Tecnologías

- **React 18** - Biblioteca JavaScript para construir interfaces de usuario
- **TypeScript** - Superset de JavaScript con tipado estático
- **Vite** - Herramienta de build rápida y moderna
- **Tailwind CSS** - Framework CSS utility-first para diseño rápido
- **Lucide React** - Biblioteca de iconos modernos

## 📦 Instalación

1. Clona el repositorio
2. Instala las dependencias:

```bash
npm install
```

## 🛠️ Desarrollo

Inicia el servidor de desarrollo:

```bash
npm run dev
```

El sitio estará disponible en `http://localhost:3000`

## 🏗️ Build para Producción

Crea una versión optimizada para producción:

```bash
npm run build
```
